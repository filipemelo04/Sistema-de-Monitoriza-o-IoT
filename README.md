# Sistema-de-Monitoriza-o-IoT
Projeto full-stack de monitorização ambiental IoT, integrando ESP32, Raspberry Pi, MQTT, InfluxDB e FastAPI. Os dados dos sensores são recolhidos e enviados em tempo real para uma API Python, armazenados numa base de dados de séries temporais e apresentados numa página web interativa construída em HTML, CSS e JavaScript com gráficos Chart.js.
